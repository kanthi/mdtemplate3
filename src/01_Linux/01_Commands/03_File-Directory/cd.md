# cd 

change directory 


**Common Usage :**  ::

		$ cd <path-to-directory>

		
**Examples :**

.. code-block:: bash

	
		king@sysadminlabs:~$ pwd
		/home/king
	
		king@sysadminlabs:~$ cd /tmp
	
		king@sysadminlabs:/tmp$ cd ..
	
		king@sysadminlabs:/$ pwd
		/
	
		king@sysadminlabs:/$ cd /tmp
	
		king@sysadminlabs:/tmp$ pwd
		/tmp
